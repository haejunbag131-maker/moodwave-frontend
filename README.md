# MOOD WAVE

음악 스트리밍 개인화 추천 웹 서비스

---

## 소개

MOOD WAVE는 음악 검색 / 재생뿐 아니라  
AI 감정 분석, 날씨 기반 추천, 음악 취향 분석 기능을 제공하는 웹 서비스입니다.

---

## 팀 구성

- 박해준 (Front-End)
- 송동현 (Front-End)
- 박소연 (Front-End)
- 이수아 (Front-End)

---

## 기술 스택

- HTML / CSS / JavaScript
- MySQL + Spring Boot
- YouTube Data API v3
- YouTube IFrame API
- OpenAI API
- OpenWeather API
- Chart.js
